# Калькулятор-бот

Тупой телеграм-бот, необходимый доказать моё превосходство над одним человеком. Основная логика написана в метро за 3 остановки где-то: (Выборгская -> Площадь Мужества, метро СПб)

Главная идея бота решать уровнения с одним неизвестным, но с любыми степенями неизвестных. Сам решатель написан не мной, я использую [SymPy](https://www.sympy.org/), вопрос в чём ценность этого изобретения до сих пор неизвестен. По мимо этого функционала здесь есть ещё что-то, но описывать этого смысла нету, считайте это отсылками.

Функционал же телегам бота реализован по трушному через [aiogram](https://aiogram.dev/).

# Запуск

```cmd
# установить зависимости
pip install -r requirements.txt

# сохранить токен бота в `token.txt`

python bot.py
```

# Зависимости

- aiogram
- sympy
